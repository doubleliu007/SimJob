# SimJob - 趣味模拟面试

一个面向求职者的趣味性网站，通过 AI 多角色扮演模拟真实的面试全流程——从简历筛选到部门面试再到董事长终面。

## 功能

### 三轮面试流程

1. **简历筛选讨论** — 多位公司高管和 HR 在会议室里讨论你的简历，每位都有独特性格，最终由总经理拍板是否通过
2. **部门面试** — 部门主管和 HR 轮番提问，根据你的回答灵活追问，面试结束后各面试官独立评估并给出通过/不通过结论
3. **董事长终面** — 副总裁和董事长亲自出马，聊格局、聊价值观，最终决定是否发放 Offer

### 核心特色

- **多 Agent 协作**：每轮面试由多个 AI 角色协同完成，各角色有独立性格（温和、冷淡、热情、毒舌等），表现各异
- **职位描述（JD）支持**：可以粘贴目标岗位的 JD，让面试官的提问和评估更有针对性；不填则自动生成一份简短 JD
- **真实公司模拟**：支持选择行业和公司，AI 会根据公司实际情况（业务方向、企业文化等）进行面试
- **语音输入**：支持浏览器语音识别，模拟真实面试体验
- **简历优化建议**：简历筛选结束后，资深 HR 会给出具体的简历和自我介绍优化建议
- **纯前端运行**：所有数据保存在浏览器 localStorage 中，不上传到任何服务器

## 技术栈

- React 19 + TypeScript + Vite 7
- TailwindCSS v4
- Zustand（状态管理）
- OpenAI JS SDK（兼容第三方中转站）

## 使用方式

```bash
npm install
npm run dev
```

打开 http://localhost:5173，在设置页配置你的 API Key、Base URL 和模型名称即可开始。

## 面试流程

```
准备材料（简历、自我介绍、目标岗位、JD）
  ↓
简历筛选讨论 → 通过？
  ↓ 是
部门面试 → 通过？
  ↓ 是
董事长终面 → 发放 Offer 🎉
```

## 注意事项

- 纯前端运行，数据不会上传到任何服务器
- 需要 OpenAI 兼容的 API 接口（支持第三方中转站）
- 对话记录保存在浏览器 localStorage 中
- 支持导出会议/面试记录为文本文件
